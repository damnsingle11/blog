五月不断电

发表于 2015-05-04 | 分类于 打油诗文 |

五月不断电

是我的妄念

裹着被子太热

光着大腿难睡

颠簸的公交是老年的摇椅

摇摇晃晃迷蝶的南柯一梦

你是我遥远的故乡

近乡情怯孤夜天狼

五月的第一场雨

惊起了波澜涟涟

梦烤熟了肉片

雨打开你的伞

我打碎了碗

几场清雨灌了安眠药

发表于 2015-04-19 | 分类于 打油诗文 |

几场清雨灌了安眠药

昏昏沉沉无聊到睡觉

开着电脑踱来踱去不知玩什么好

无意输入的字母成了熟悉的名字

想起四角天空下那间教室

开小差拿着笔在书上乱写

或陌上花开春风十里

又郎骑竹马明日天涯

如此般矫情做作而自恋：此字甚好！

有时无意间写下的竟是你的名字

却立即毁尸灭迹涂成厚厚小圆点

像电脑上突然地蹦出的一个黄页

我带着不纯洁的负罪感点下红叉

生怕周围的人瞥见我的猫腻

你的名字是我字典里的禁忌

黑夜与白昼差着枕头

关山和长江差着车流

拥堵的车流载不动枕间的绮梦

我与世界只差一个你

春风与夏蝉差着温度

钟楼和后林差着月华

冰冷的月光难沁彻想念的温润

我与世界只差一个你

信笺与工笔差着诗意

手机和字符差着素情

忠实的机器显不了笔尖的迷漾

我与世界只差一个你

生活就要写成了时间表

朝九晚五

埋葬了朝三暮四

穿衣，洗漱，吃饭，上班

重复这一生要学会的循环

在出门之前把自己变成利剑

因为下一刻就要为信仰而战

走进射鸡湿这个坑

便关注街上的花红柳绿的广告

就像自从认识了你

眼睛害怕城市人头攒动的熟悉

同在这座城
遇到的却是不同的人

也许是这个夏天来得太快

本该平静的心也躁动起来

可我还不知道归往何处

才能找到延续信仰的神

我敬佩那群六人一队的小兵

发表于 2015-03-13 | 分类于 打油诗文 |

我敬佩那群六人一队的小兵

看到每天死亿万同胞惨死连眼泪都不许流

存在的意义就是被杀掉变成金币

死了连尸体都很快消失

长期从事高强度（对抗英雄，拆防御塔）

高风险（随时都会挂）的工作

却无法获得任何实际上的报酬

反而要将毕生积蓄拱手送人

就这样还得勇往直前

从不像英雄卑劣的偷袭和逃跑

从不像人类那样抱怨“生活已经如此艰难”

即使是面对比自己强百倍的敌人

依然义无反顾的选择一条路

清楚的知道自己的目标

也不会刻意回避地前行

把头摁进被褥

发表于 2015-03-12 | 分类于 打油诗文 |

把头摁进被褥

就拽出一个黎明

时间太快，来不及做一场梦

梦一场皓月，把思念丢给月，不知要反射多久，才能照进你的窗棂。

梦一场春雨，突来起来的雨，最美不是下雨天，是你躲雨下的屋檐。

梦一场人海，翻涌滚动的海，二百五十的近视，一眼找到生命的舟。

三月的一切都是新的，从手到脚。

以梦为马，策马扬鞭。
